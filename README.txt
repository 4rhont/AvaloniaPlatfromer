# Avalonia Platformer

2D платформер, разрабатываемый на Avalonia UI с использованием ReactiveUI и MVVM архитектуры.

## Текущий статус
- Базовая структура проекта
- Физика движения и прыжков  
- Управление через WASD/стрелки
- ReactiveUI привязки
- Добавлены платформы
- Добавлена физика коллизий

## Глобальные цели
- Добавление камеры
- Добавление основных механик (атака, виды атаки)
- Добавление предметов, инвентаря
- Добавление меню, настроек
- Добавление противников, эффектов состояния персонажа
- Прорабатывание уровней, боссов

- Отрисовка моделек
- Отрисовка спрайтов 
- Отрисовка фонов
- Добавление многослойного фона
- Добавление эффектов освещения, виньетки

- Анализ получившейся игры и добавление новых механик
